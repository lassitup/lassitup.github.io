CSPB 2270 Data Structures Final Project - Financial Security Pledging Application

Attached is a link to my Github repository containing the results of my semester project for my CSPB 2270 course (Data Structures) Fall 2024 semester Final Semester Project for CSPB 2270 Data Structures (Fall 2024). For this project, the class was tasked with choosing a data structure we have no used in class and come up with our own idea on how to use the structure. My project idea was creating an application to automate the security pledging process I perform daily in my job as a financial analyst. For this project, I implemented a Red-Black Tree to store and organize financial security data allowing for max search time complexity of O(logN). The program then performs a series of algorithms to ensure customer balances are sufficiently pledged and will export the change in securities pledged. Highlights of this project was the use of the red-black tree structure and heavy usage of pointer nodes for each security, customer and account. CSPB 2824 RSA Cryptosystem- Discrete Mathematics Project (Python)

Attached is a link to my Github repository containing the results of my semester project for my CSPB 2824 course (Discrete Structures) Summer 2024 semester The objective of this project was to create a Jupyter Notebook explaining in detail how the RSA Cryptosystem works and why it is secure. Within the project, I walk throught the necessary math to make the encryption / decryption scheme possible such as modular inverses, greatest common divisors, modular exponentiation, Bezout Coefficients and the algorithms / python functions used to apply these mathematical concepts such as the Euclidean Algorithm and Fast Modular Exponentiation.
